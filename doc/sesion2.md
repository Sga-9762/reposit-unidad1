# Resumen Sesión 2
## ¿Cómo se crea un repositorio nuevo en GitHub?
#### Para crear un repositorio en GitHub, utilizando nuestra cuenta registrada. debemos navegar a la pagina web github.com, allí creamos un nuevo repositorio desde la pestaña de 'crear'. Introducir datos como nombre y descripción. Este será nuestro repositorio en la nube, en el que se podrán ver los archivos enlazados desde el repositorio local.

## ¿Cómo se enlaza con el repositorio local? 

#### Para enlazar la carpeta o repositorio local al repositorio de la nube en GitHub, se utilizan los comandos 'git remote add origin', 'git remote -v' y 'git push'. En el caso del comando 'git remote add origin', este va seguido del enlace URL del repositorio en la nube, Estos comandos permiten subir los archivos ubicados en el repositorio local al repositorio en la nube de GitHub.
